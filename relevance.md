# Актуальность

В условиях работы с большими массивами данных существует вероятность пропуска каких-либо значений. Следовательно, для сохранения целостности данных необходимо выявлять данные пропуски и восстанавливать их таким образом, чтобы восстановленные данные не выбивались из общей структуры.
